# PDF‑to‑E‑Course Learning Platform

A full‑stack AI‑powered platform that converts uploaded PDFs into structured e‑courses.

- **Frontend** – React 19, Vite, Tailwind CSS, React Router, Axios, React Query, React Hook Form, Framer Motion, Lucide Icons, optional shadcn/ui.
- **Backend** – Node.js, Express, JWT, Multer, pdf‑parse, bcrypt, Mongoose (MongoDB Atlas).
- **AI** – Groq API (fallback OpenRouter).
- **Deployment** – Frontend on Vercel, Backend on Render.

The repository follows a clean‑architecture layout and includes CI, testing, and production‑ready scripts.
